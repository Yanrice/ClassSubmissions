Weather analysis Latitude



Visualizations



we created a visualisation dashboard about the weather of more than five hundred cities across the world of varying distance from the equator (lat_range = (-90, 90), lng_range = (-180, 180)). To accomplish that we utilised python library(https://pypi.python.org/pypi/citipy), API and Json traversals to collect and analyse data from openweather API (https://openweathermap.org/api).


After creating a representative model of weather across world cities using using a Jupyter notebook, we use Matplotlib, we built a serie of scatter plots to showcasing relationships between latitude and temperature (F), humidity (%), cloudiness (%), wind speed (mph).
